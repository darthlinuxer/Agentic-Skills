### Category 2: Need Sequential Thinking (Complex Reasoning)
Skills involving multi-step problem solving or complex processes:

1. **architecture** - System design requires step-by-step thinking
2. **systematic-debugging** - Root cause analysis is inherently sequential
3. **problem-solving** - Meta-level problem decomposition
4. **game-development** - Complex game logic and mechanics
5. **game-design** - Balancing, progression systems
6. **security-auditor** - Threat modeling and analysis
7. **red-team-tactics** - Attack chain planning
8. **database-design** - Schema design and optimization
9. **subagent-driven-development** - Multi-agent coordination
10. **parallel-agents** - Agent orchestration

### Category 3: Need Both (Highly Complex)
Skills requiring both updated docs AND complex reasoning:

1. **app-builder** - Full-stack requires both
2. **frontend-design** - Design thinking + current trends
3. **backend-development** - Architecture + current patterns
4. **performance-optimizer** - Analysis + current tools
5. **brainstorming** - Complex ideation + research
6. **research** - Deep analysis + documentation search
7. **plan-writing** - Strategic planning + current methodologies
