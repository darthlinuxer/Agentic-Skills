# Visão geral

O Termo de Abertura do Projeto (Project Charter) autoriza formalmente o início do projeto, estabelece a autoridade do Gerente do Projeto e define objetivos, escopo em alto nível, benefícios esperados e principais stakeholders. Este artefato marca a transição de uma ideia para um projeto autorizado.

## Alinhamento com PMBOK 7

O Project Charter é um **Strategy Artifact** que suporta especialmente os Performance Domains de **Stakeholders** e **Planning**. Ele define a base de governança e orienta decisões iniciais.

## Momento de elaboração

Deve ser desenvolvido durante a **iniciação** do projeto, após aprovação do Business Case (quando aplicável) e antes do planejamento detalhado.

## Responsabilidade

Normalmente é produzido pelo patrocinador ou iniciador do projeto, em colaboração com o Gerente do Projeto designado.
