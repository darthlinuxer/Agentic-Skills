# Histórico de versões dos anexos

## Objetivo

Registrar a evolução dos principais anexos para garantir rastreabilidade e transparência das mudanças.

## Campos recomendados

- Documento
- Versão
- Data
- Mudanças realizadas
- Autor/Responsável

## Boas práticas

- Atualizar a cada nova versão aprovada.
- Registrar apenas mudanças relevantes.
- Indicar relação com solicitações de mudança ou baselines, quando aplicável.
