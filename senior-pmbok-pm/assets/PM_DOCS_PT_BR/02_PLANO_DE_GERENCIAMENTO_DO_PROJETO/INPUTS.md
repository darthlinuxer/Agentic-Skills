# INPUTS — Plano de Gerenciamento do Projeto (PMP)

Este arquivo mapeia todas as entradas necessárias para preencher o `TEMPLATE.md`.

## Como usar

Para cada chave abaixo, forneça o valor correspondente. Onde indicado como lista/tabela, use o formato sugerido.
