# 7. Autoridade do Gerente do Projeto

Defina claramente a autoridade delegada ao GP para decisões operacionais, orçamento e gestão de pessoal, além de critérios de escalonamento.

## 7.1 Autoridade para Decisões

Especifique o que o GP pode aprovar sem escalonamento (escopo, orçamento, cronograma, riscos, design etc.).

## 7.2 Decisões que Requerem Escalonamento

Defina quando o GP deve escalonar para patrocinador ou governança.

## 7.3 Gestão de Orçamento

Declare orçamento total, reservas e limites de aprovação.

## 7.4 Gestão de Pessoal

Indique autoridade para requisitar, avaliar e aprovar ações relacionadas à equipe.
