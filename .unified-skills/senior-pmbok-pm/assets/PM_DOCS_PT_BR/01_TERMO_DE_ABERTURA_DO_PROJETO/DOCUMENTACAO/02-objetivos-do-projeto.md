# 2. Objetivos do Projeto

Objetivos devem ser **SMART** (Specific, Measurable, Achievable, Relevant, Time-bound) e focados no **resultado** que o projeto entregará.

## 2.1 Objetivo Geral

Declaração de alto nível do objetivo principal do projeto, com prazo e benefício esperado.

## 2.2 Objetivos Específicos (SMART)

Liste entre 3 e 7 objetivos mensuráveis com métricas, metas, prazos e método de medição.

## 2.3 Critérios de Validação dos Objetivos

Defina critérios para verificar se cada objetivo atende aos requisitos SMART.
