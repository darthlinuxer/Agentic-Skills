# Registro de issues

## Objetivo
Rastrear impedimentos e problemas relevantes, garantindo resolução com prioridade adequada.

## Campos recomendados
- ID
- Descrição
- Categoria
- Impacto
- Prioridade
- Responsável
- Status

## Ritmo de acompanhamento
- Revisão semanal ou conforme criticidade.
- Escalonamento quando necessário.
