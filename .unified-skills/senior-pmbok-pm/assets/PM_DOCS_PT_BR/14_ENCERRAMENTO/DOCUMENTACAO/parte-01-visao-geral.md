# Visão geral

## Propósito
Formalizar aceitação final, transferir produtos/serviços, capturar lições aprendidas, encerrar contratos, liberar recursos e arquivar documentação.

## Alinhamento PMBOK
- Delivery: formalizar entregas.
- Stakeholders: obter aceitação final.
- Team: reconhecimento e liberação.
- Measurement: base para benefícios.
