# Relatórios e dashboards

## Princípios
- Adequar formato e detalhe à audiência.
- Visualização clara (RAG status, tendências).
- Métricas acionáveis.

## Tipos de report
- Executive dashboard (mensal).
- Status report (semanal/quinzenal).
- Variance report (mensal).
- Risk report (quinzenal).

## Dashboards
- Executivo (alto nível).
- Gerencial (EVM e tendências).
- Técnico (builds, testes, bugs).
