# Lista de documentos do projeto

## Objetivo do inventário

O inventário de documentos organiza as principais entregas documentais por fase do projeto e garante rastreabilidade. Deve permitir localizar rapidamente cada documento e sua versão vigente.

## Estrutura por fase

Recomenda-se categorizar por:
- **Iniciação**
- **Planejamento**
- **Execução e controle**
- **Técnicos**
- **Qualidade**
- **Encerramento**

## Campos recomendados

- ID do documento
- Nome do documento
- Localização (repositório/pasta/ferramenta)
- Versão
- Data
- Responsável (owner)
- Status (opcional)

## Boas práticas

- Padronizar IDs (ex.: DOC-001).
- Referenciar links permanentes.
- Atualizar sempre que houver mudança de versão.
- Indicar responsáveis por manutenção e aprovação.
