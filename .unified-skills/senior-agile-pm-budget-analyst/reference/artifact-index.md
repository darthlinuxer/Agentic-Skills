## Artifact selection rules
1. **Identify language first** (PT-BR or EN) based on the user request.
2. **Exact match** on folder name or canonical artifact name → select directly in the language-specific base path.
3. **Alias match** (see table) → select the mapped PT-BR folder, then use the mirrored EN path if needed.
4. **Ambiguous** → ask the user which artifact they mean.
5. **Multiple artifacts** → for complete project analysis, typically use: Iniciativas → Épicos → User Stories → Poker Planning → Orçamento → Gantt → Caminho Crítico

## Required files per artifact
- **Template**: `TEMPLATE.md` (source of structure)
- **Inputs**: `INPUTS.md` (required data)
- **Documentation**: `DOCUMENTACAO.md` or `DOCUMENTACAO/` (guidance, best practices, and constraints)

## Typical workflow sequences

### Full Project Analysis
1. Start with `10_ANALISE_PROJETO` (overall project breakdown)
2. Create `01_INICIATIVAS` (strategic initiatives)
3. Break into `02_EPICOS` (epics per initiative)
4. Detail `03_USER_STORIES` (cards per epic)
5. Estimate with `04_POKER_PLANNING` (story points)
6. Prioritize `05_BACKLOG_PRIORIZADO` (prioritized backlog)
7. Calculate `07_ORCAMENTO_PROJETO` (budget based on points)
8. Generate `08_GANTT_CHART` (timeline)
9. Analyze `09_CAMINHO_CRITICO` (critical path)

### Sprint Planning
1. Review `05_BACKLOG_PRIORIZADO` (top priority items)
2. Execute `06_SPRINT_PLANNING` (select stories for sprint)
3. Reference `04_POKER_PLANNING` (validate estimates)
4. Track with `11_VELOCITY_BURNDOWN` (monitor progress)

### Retrospective & Improvement
1. Conduct `12_RETROSPECTIVA` (sprint retrospective)
2. Update `11_VELOCITY_BURNDOWN` (adjust velocity)
3. Refine `05_BACKLOG_PRIORIZADO` (reprioritize based on learnings)

## Configuration parameters (ask user at start)
- **Poker planning ratio**: Default 5 points = 2 weeks (1 sprint)
- **Team composition**: Roles and quantities
- **Hourly/daily rates**: Per role
- **Sprint duration**: Typically 2 weeks
- **Team velocity**: Points per sprint (if known)
- **Currency**: For budget calculations
