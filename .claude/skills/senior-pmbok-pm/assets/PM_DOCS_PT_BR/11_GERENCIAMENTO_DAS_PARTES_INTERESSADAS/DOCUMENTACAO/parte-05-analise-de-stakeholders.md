# Análise de stakeholders

## Dimensões principais
- **Poder/Influência**: capacidade de impactar decisões.
- **Interesse**: nível de envolvimento/impacto.
- **Impacto se desengajado**: risco para o projeto.
- **Atitude**: resistente → neutro → apoiador → líder.

## Mapeamentos
- **Power x Interest Grid**: define estratégia por quadrante.
- **Assessment Matrix**: atual vs desejado.
- **Salience Model**: poder + legitimidade + urgência.

## Estratégias por quadrante
- **Manage Closely**: alto poder e alto interesse.
- **Keep Satisfied**: alto poder, baixo interesse.
- **Keep Informed**: baixo poder, alto interesse.
- **Monitor**: baixo poder, baixo interesse.

## Conflitos e expectativas
- Documentar expectativas explícitas e implícitas.
- Mapear conflitos entre stakeholders e necessidades do projeto.
- Definir ações para reduzir gaps de engajamento.
