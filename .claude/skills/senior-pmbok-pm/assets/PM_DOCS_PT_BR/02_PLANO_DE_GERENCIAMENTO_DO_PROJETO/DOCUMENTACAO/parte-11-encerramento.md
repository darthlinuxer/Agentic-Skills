# 11. Encerramento do Projeto

Define critérios e atividades para concluir o projeto, formalizar aceitação e capturar lições aprendidas.

## 11.1 Critérios para Encerramento

Liste condições necessárias para encerrar o projeto: entregas aceitas, objetivos atingidos ou desvios aceitos, transição concluída, contratos encerrados, documentação arquivada e aprovação formal.

## 11.2 Atividades de Encerramento

Descreva passos como aceitação final, encerramento contratual, transição, lições aprendidas, avaliação de desempenho, realização de benefícios, arquivamento e liberação de recursos.
