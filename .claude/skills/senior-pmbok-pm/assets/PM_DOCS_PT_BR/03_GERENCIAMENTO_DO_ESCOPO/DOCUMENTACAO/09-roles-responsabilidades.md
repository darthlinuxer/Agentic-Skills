# 9. Roles e Responsabilidades — Gerenciamento do Escopo

Define responsabilidades por papel para garantir governança e qualidade na gestão do escopo.

**Boas práticas**:
- Nomear responsáveis por cada processo (definição, validação, controle).
- Garantir que decisões finais de priorização estejam explícitas.
- Prever papéis de apoio (BA, QA, PMO, arquiteto) conforme o contexto.
