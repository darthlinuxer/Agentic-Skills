 # Create Skill Reference
 
 This reference expands the core `create-skill` guidance with checklists and templates.
 
 ## Minimal Skill Checklist
 - Frontmatter includes `name` and `description`
 - Clear “when to use” triggers
 - Step-by-step instructions
 - One concise example
 - No unnecessary external dependencies
 
 ## Structure Template
 ```
 ---
 name: skill-name
 description: Use when...
 ---
 
 # Skill Title
 
 ## When to Use
 - Trigger terms
 - Typical scenarios
 
 ## Instructions
 1. Step one
 2. Step two
 
 ## Example
 [Short, concrete example]
 ```
