 # Skill Standards
 
 ## Naming
 - Lowercase letters, numbers, hyphens only
 - Max 64 characters
 - Unique and specific
 
 ## Descriptions
 - Start with “Use when…”
 - Include what the skill does and when to apply it
 - Avoid vague wording
 
 ## Content
 - Prefer concise instructions
 - Keep SKILL.mdc under 500 lines
 - Use progressive disclosure for long references
