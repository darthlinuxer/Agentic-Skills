# Entregas planejadas vs realizadas

## Objetivo
Controlar o andamento das entregas, comparar planejado x realizado e tratar variações.

## Práticas
- Registro consolidado de entregas.
- Comparação periódica (planejado/real/variação).
- Ações corretivas para desvios.

## Indicadores sugeridos
- % entregas no prazo.
- Desvio médio (dias/sprints).
- Lead time por entrega.
