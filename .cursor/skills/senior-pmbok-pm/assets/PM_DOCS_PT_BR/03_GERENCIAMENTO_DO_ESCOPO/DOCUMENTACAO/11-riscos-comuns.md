# 11. Riscos Comuns no Gerenciamento do Escopo

Esta seção lista riscos típicos e orientações para mitigação.

**Boas práticas**:
- Monitorar indicadores de scope creep e volatilidade de requisitos.
- Conduzir validações incrementais para reduzir retrabalho.
- Formalizar o processo de mudanças e educar stakeholders.
